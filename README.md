## Getting Started

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

## Bun Version

This project was created using `bun init` in bun v1.1.27. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
